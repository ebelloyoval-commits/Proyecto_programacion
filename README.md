# Calculadora Básica

## Descripción General

El presente proyecto consiste en el desarrollo de una calculadora básica implementada en Python, capaz de realizar operaciones matemáticas fundamentales como suma, resta, multiplicación y división. El sistema permite al usuario ingresar dos valores numéricos y seleccionar la operación deseada, mostrando el resultado de manera clara en pantalla.

## Objetivo

El objetivo principal del proyecto es aplicar los conocimientos de programación para desarrollar un sistema funcional sencillo, demostrando el uso de estructuras básicas como entrada de datos, operaciones aritméticas, condicionales y salida de resultados.

## Funcionamiento

El programa solicita al usuario que introduzca dos números y seleccione la operación que desea realizar. Dependiendo de la opción elegida, el sistema ejecuta la operación correspondiente y muestra el resultado. En el caso de la división, el sistema puede incluir validaciones para evitar errores como la división entre cero.

## Características

* Interfaz basada en consola
* Operaciones básicas: suma, resta, multiplicación y división
* Entrada de datos por el usuario
* Resultados mostrados en pantalla
* Código simple y fácil de entender

## Estructura del Proyecto

El proyecto está organizado en diferentes carpetas para mantener un orden adecuado:

* src/: contiene el código fuente del programa
* docs/: incluye evidencia del funcionamiento del sistema (capturas de pantalla)

## Tecnologías Utilizadas

* Lenguaje de programación: Python
* Entorno de ejecución: consola del sistema

## Resultados

Se obtuvo un programa funcional capaz de realizar operaciones matemáticas básicas correctamente, cumpliendo con los requerimientos de una versión inicial del sistema.

## Conclusión

El desarrollo de esta calculadora permitió comprender y aplicar conceptos fundamentales de programación, así como la organización de un proyecto en un repositorio. Este sistema puede ser mejorado en futuras versiones agregando más funcionalidades o una interfaz gráfica.
